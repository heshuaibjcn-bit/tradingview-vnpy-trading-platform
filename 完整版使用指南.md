## 📖 快速开始

### 第一步：打开界面

```bash
# 浏览器访问
http://localhost:8080
```

### 第二步：熟悉TradingView

完整的TradingView界面，包括：

**顶部工具栏：**
- 品种切换（点击股票代码）
- 周期选择（1分、5分、日线、周线等）
- 指标按钮（添加技术指标）
- 画线工具（趋势线、支撑阻力等）
- 设置按钮

**左侧面板：**
- 画线工具箱
- 研究面板（技术指标）
- 观察列表
- 详情面板

**右侧面板：**
- 详情/财经日历
- 股票筛选器
- 聊天室（可选）

**底部面板：**
- 时间周期选择器

### 第三步：打开交易插件

1. 点击右上角的 **📈 蓝色插件按钮**
2. 交易面板从右侧滑入
3. 面板包含4个选项卡：
   - **账户** - 账户信息和实时行情
   - **交易** - 快速下单表单
   - **持仓** - 当前持仓列表
   - **订单** - 活跃订单管理

### 第四步：登录账户（如需要）

1. 在"账户"选项卡查看状态
2. 如未登录，会自动弹出登录框
3. 输入用户名和密码
4. 登录成功后加载账户数据

### 第五步：开始交易

1. 切换到"交易"选项卡
2. 选择方向：[买入] 或 [卖出]
3. 输入股票代码（如：600000）
4. 输入价格（自动填充当前价）
5. 输入数量（100股倍数）
6. 点击"提交订单"

## 🎨 界面说明

### 完整TradingView功能

#### 1. 图表操作

| 操作 | 方法 |
|------|------|
| 缩放 | 鼠标滚轮 |
| 平移 | 鼠标拖拽 |
| 十字光标 | 点击图表 |
| 查看详情 | 鼠标悬停K线 |

#### 2. 技术指标

**添加指标：**
1. 点击顶部"指标"按钮
2. 搜索指标名称（如：MA、RSI、MACD）
3. 点击添加

**常用指标：**
- MA (移动平均线)
- EMA (指数移动平均)
- RSI (相对强弱指标)
- MACD (指数平滑异同移动平均线)
- Bollinger Bands (布林带)
- Volume (成交量)
- 以及100+其他指标

#### 3. 画线工具

**打开工具箱：**
- 点击左侧画线图标

**常用工具：**
- 趋势线
- 支撑/阻力线
- 斐波那契回调
- 矩形、圆形等形状
- 文字标注

#### 4. 品种切换

**方法1：** 点击顶部股票代码，输入新代码

**方法2：**
- 点击左侧"观察列表"
- 选择已保存的品种

**支持品种：**
- A股股票（如：SSE:600000）
- 港股（如：HKEX:00700）
- 美股（如：NASDAQ:AAPL）
- 期货、外汇、加密货币等

### 交易插件功能

#### 选项卡1: 账户

**账户信息：**
- 总资产 - 所有资产总和
- 可用资金 - 可用于交易的资金
- 持仓市值 - 当前持仓总市值
- 今日盈亏 - 当日盈亏金额

**实时行情：**
- 最新价 - 当前最新成交价
- 涨跌幅 - 当日涨跌百分比
- 成交量 - 当日成交量

#### 选项卡2: 交易

**快速下单：**
```
┌────────────────────┐
│ [买入] [卖出] ← 选择方向 │
├────────────────────┤
│ 股票代码: [600000] │
│ 价格: [10.52]      │
│ 数量: [1000]       │
├────────────────────┤
│ [提交订单]         │
└────────────────────┘
```

**字段说明：**
- 方向：买入(做多) 或 卖出(做空)
- 股票代码：6位数字代码
- 价格：限价单价格
- 数量：股数（必须是100的倍数）

**验证规则：**
- ✅ 价格必须大于0
- ✅ 数量必须是100的倍数
- ✅ 需要登录才能下单

#### 选项卡3: 持仓

**显示信息：**
- 股票代码
- 方向标签（多/空）
- 持仓数量
- 持仓价格

**颜色标识：**
- 绿色 多头仓位
- 红色 空头仓位

#### 选项卡4: 订单

**显示信息：**
- 订单ID
- 股票代码
- 方向（买入/卖出）
- 数量和价格
- 订单状态

**订单状态：**
- 未成交
- 部分成交
- 已成交
- 已撤销

**操作：**
- 点击"撤单"按钮撤销订单

- 价格自动填充：开启/关闭

1. 点击顶部股票代码
2. 输入多个品种，用逗号分隔
3. 例如：`600000,000001,600036`
4. TradingView会同时显示多个品种

## ⚠️ 注意事项

### TradingView加载

**首次访问可能需要：**
- 10-15秒加载时间
- 加载TradingView Widget库
- 初始化图表组件

**如果图表不显示：**
1. 检查网络连接
2. 等待更长时间（最多30秒）
3. 刷新页面（Ctrl+F5）
4. 检查浏览器控制台错误

### 数据延迟

**行情数据：**
- 实时WebSocket推送
- 延迟通常<100ms
- 受网络状况影响

**账户数据：**
- 登录后自动加载
- 下单后自动更新
- 可手动刷新（F5）

### 交易插件

**插件不显示：**
- 检查是否点击了插件按钮
- 插件在右侧，可能被遮挡

**数据不更新：**
- 检查连接状态（左下角）
- 刷新页面重新连接
- 检查后端服务是否运行
