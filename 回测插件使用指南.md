# 回测插件使用指南

**版本**: v6.1 - 回测系统
**完成日期**: 2026-03-06

---

## 🎯 功能概述

回测插件是TradingView量化交易系统的重要组成部分，允许用户在历史数据上测试交易策略，评估策略表现，优化策略参数。

### 核心功能

✅ **策略回测**
- 多种预置策略（均线交叉、突破、动量）
- 自定义策略支持
- 历史数据回放

✅ **绩效分析**
- 总收益率、年化收益率
- 最大回撤、夏普比率
- 胜率、交易次数

✅ **交易记录**
- 详细交易历史
- 买卖点位展示
- 盈亏统计

✅ **参数优化**
- 可调策略参数
- 多种回测期间
- 灵活的初始资金设置

## 🚀 快速开始

### 1. 打开回测界面

```
浏览器访问: http://localhost:8080
```

1. 点击右上角 📈 插件按钮
2. 切换到"回测"选项卡

### 2. 设置回测参数

**基本参数:**
- **策略类型**: 选择回测策略
- **股票代码**: 输入6位股票代码（如：600000）
- **起始日期**: 回测开始日期
- **结束日期**: 回测结束日期
- **初始资金**: 初始投入资金（默认¥1,000,000）

**策略参数:**

**均线交叉策略:**
- 短期均线：默认5天
- 长期均线：默认20天

**突破策略:**
- 突破周期：默认20天
- 突破幅度：默认2%

**动量策略:**
- 动量周期：默认10天
- 动量阈值：默认2%

### 3. 运行回测

1. 设置好所有参数
2. 点击"运行回测"按钮
3. 等待回测完成（通常3-10秒）
4. 查看回测结果

### 4. 查看结果

**绩效指标:**
- 总收益率：策略总体收益
- 年化收益率：年化平均收益
- 最大回撤：最大亏损幅度
- 夏普比率：风险调整后收益
- 胜率：盈利交易占比
- 交易次数：总交易笔数

**交易记录:**
- 交易日期
- 买卖方向
- 成交价格
- 成交数量

## 📊 策略说明

### 1. 均线交叉策略 (MA Cross)

**原理:**
- 短期均线上穿长期均线（金叉）→ 买入信号
- 短期均线下穿长期均线（死叉）→ 卖出信号

**参数:**
- `short_ma`: 短期均线周期（建议5-10天）
- `long_ma`: 长期均线周期（建议20-60天）

**适用场景:**
- 趋势明显的市场
- 中长期投资
- 震荡市表现较差

**示例配置:**
```
短期均线: 5天
长期均线: 20天
回测期间: 2024-09-06 到 2026-03-06
```

### 2. 突破策略 (Breakout)

**原理:**
- 价格突破N日最高价 → 买入信号
- 价格跌破N日最低价 → 卖出信号

**参数:**
- `breakout_period`: 突破周期（建议20天）
- `breakout_pct`: 突破幅度（建议1-3%）

**适用场景:**
- 波动较大的市场
- 趋势启动阶段
- 需要设置止损

**示例配置:**
```
突破周期: 20天
突破幅度: 2%
回测期间: 2024-09-06 到 2026-03-06
```

### 3. 动量策略 (Momentum)

**原理:**
- 价格涨幅超过阈值 → 买入信号
- 价格跌幅超过阈值 → 卖出信号

**参数:**
- `momentum_period`: 动量周期（建议5-15天）
- `momentum_threshold`: 动量阈值（建议1-5%）

**适用场景:**
- 强势股追涨
- 短期交易
- 需要快进快出

**示例配置:**
```
动量周期: 10天
动量阈值: 2%
回测期间: 2024-09-06 到 2026-03-06
```

## 📈 绩效指标解读

### 收益指标

**总收益率**
```
总收益率 = (最终资产 - 初始资金) / 初始资金
```
- > 20%: 优秀
- 10%-20%: 良好
- 0%-10%: 一般
- < 0%: 亏损

**年化收益率**
```
年化收益率 = (最终资产 / 初始资金)^(1/年数) - 1
```
- > 30%: 优秀
- 15%-30%: 良好
- 5%-15%: 一般
- < 5%: 较差

### 风险指标

**最大回撤**
```
最大回撤 = (峰值 - 谷值) / 峰值
```
- < 10%: 低风险
- 10%-20%: 中等风险
- 20%-30%: 高风险
- > 30%: 极高风险

**夏普比率**
```
夏普比率 = (年化收益率 - 无风险利率) / 波动率
```
- > 2: 优秀
- 1-2: 良好
- 0.5-1: 一般
- < 0.5: 较差

### 交易指标

**胜率**
```
胜率 = 盈利交易次数 / 总交易次数
```
- > 60%: 高胜率
- 50%-60%: 中等胜率
- 40%-50%: 低胜率
- < 40%: 需要优化

**交易次数**
- 过多（>100）: 可能过度交易
- 适中（20-100）: 合理范围
- 过少（<20）: 样本不足

## 🎯 使用建议

### 回测流程

1. **策略选择**
   - 根据市场环境选择策略
   - 趋势市 → 均线交叉
   - 震荡市 → 突破策略
   - 强势股 → 动量策略

2. **参数设置**
   - 使用默认参数开始
   - 逐步优化参数
   - 避免过度优化

3. **期间选择**
   - 至少6个月数据
   - 包含牛熊市周期
   - 最近1-2年为佳

4. **结果分析**
   - 关注风险指标
   - 不要只看收益率
   - 考虑交易成本

### 参数优化

**优化步骤:**
1. 固定回测期间
2. 调整一个参数
3. 观察绩效变化
4. 记录最优参数
5. 重复步骤2-4

**注意事项:**
- 避免过度拟合
- 考虑参数稳定性
- 使用样本外验证
- 实盘前小资金测试

### 常见问题

**Q: 为什么回测收益很好，实盘却亏损？**
A: 可能原因：
- 回测数据质量问题
- 未考虑交易成本
- 滑点和成交价假设
- 过度拟合历史数据

**Q: 如何提高回测可靠性？**
A: 建议：
- 使用足够长的历史数据
- 考虑交易成本和滑点
- 进行样本外测试
- 小资金实盘验证

**Q: 哪个策略最好？**
A: 没有最好的策略，只有最适合的策略：
- 趋势市用均线策略
- 震荡市用均值回归
- 选择适合当前市场的策略

## ⚠️ 注意事项

### 数据质量

⚠️ **当前使用模拟数据**
- 回测结果基于生成的历史数据
- 不代表真实市场表现
- 仅用于演示和测试

### 策略局限

⚠️ **策略简化**
- 未考虑交易成本
- 未考虑滑点影响
- 未考虑流动性限制
- 实盘效果可能不同

### 使用建议

✅ **推荐做法:**
1. 多策略对比
2. 参数敏感性分析
3. 样本外验证
4. 小资金实盘测试
5. 严格风险控制

❌ **避免做法:**
1. 过度优化参数
2. 忽略交易成本
3. 过度杠杆
4. 盲目追求高收益
5. 忽视风险控制
