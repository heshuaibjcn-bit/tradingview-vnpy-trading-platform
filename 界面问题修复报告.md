### ✅ 问题1: 数据加载问题 (HIGH PRIORITY)

**原始问题**:
- 所有账户信息显示 "--" 占位符
- 所有实时行情显示 "--" 占位符
- 持仓列表和订单列表为空

**根本原因**:
1. **函数定义顺序错误**: `formatCurrency()` 和 `formatNumber()` 函数定义在 `loadMockData()` 之后，导致函数未定义时被调用
2. **缺少初始数据**: 未登录用户没有看到任何演示数据

**修复方案**:
1. 移动 `formatCurrency()` 和 `formatNumber()` 函数到全局变量之后，所有其他函数之前
2. 添加 `loadMockData()` 函数，为未登录用户提供模拟演示数据
3. 修改 `DOMContentLoaded` 初始化逻辑，根据token状态加载真实或模拟数据

**修复代码**:
```javascript
// 工具函数移到文件前面
const API_BASE = 'http://localhost:8080';

// ==================== 工具函数 ====================
function formatCurrency(value) {
    return '¥' + value.toFixed(2).replace(/\d(?=(\d{3})+\.)/g, '$&,');
}

function formatNumber(value) {
    return value.toString().replace(/\B(?=(\d{3})+(?!\d))/g, ',');
}

// ==================== 模拟数据加载 ====================

    };

        {

            volume: 10000,
```

### ✅ 问题2: ID冲突修复

**原始问题**:
- `id="volume"` 同时用于:
  1. 成交量显示元素
  2. 交易表单的数量输入框
- 导致JavaScript更新错误元素

**修复方案**:
将成交量显示元素ID从 `volume` 改为 `quoteVolume`

**修复代码**:
```html
<!-- 修复前 -->
<div class="quote-price" id="volume">--</div>

<!-- 修复后 -->
<div class="quote-price" id="quoteVolume">--</div>
```

```javascript
// JavaScript也相应更新
```

### ✅ 问题3: 布局比例修复 (MEDIUM PRIORITY)

**原始问题**:
- 左右比例为 75%:25% (使用 `1fr 350px`)
- 不符合设计要求的 70%:30%

**修复方案**:
将CSS grid布局从固定像素改为比例分配

**修复代码**:
```css
/* 修复前 */
.main-container {
    grid-template-columns: 1fr 350px;  /* 左侧自适应，右侧350px固定 */

    grid-template-columns: 7fr 3fr;   /* 70%:30% 比例 */
```

### ✅ 问题4: 今日盈亏字段缺失更新

**原始问题**:
- `updateAccountInfo()` 函数没有更新 `todayProfit` 元素
- 导致今日盈亏始终显示 "--"

**修复方案**:
在 `updateAccountInfo()` 函数中添加今日盈亏更新逻辑

**修复代码**:
```javascript
function updateAccountInfo(data) {
    const accounts = data.accounts || [];
    if (accounts.length > 0) {
        const account = accounts[0];
        document.getElementById('totalAssets').textContent =
            formatCurrency(account.balance || 0);
        document.getElementById('availableBalance').textContent =
            formatCurrency(account.available || 0);
        document.getElementById('positionValue').textContent =
            formatCurrency(account.frozen || 0);
        // 新增：更新今日盈亏
        document.getElementById('todayProfit').textContent =
            formatCurrency(account.pnl || 0);
    }
}
```

### ✅ 问题5: 交易表单价格自动填充 (MEDIUM PRIORITY)

**原始问题**:
- 价格输入框为空，需要用户手动输入
- 用户体验不佳

**修复方案**:
1. 添加全局变量 `currentPrice` 跟踪当前价格
2. 在 `updateQuote()` 函数中更新 `currentPrice`
3. 如果用户未手动修改过价格，自动填充当前价格
4. 添加输入监听，标记用户已修改

**修复代码**:
```javascript
// 全局变量
let currentPrice = 10.0;

// 更新行情时同时更新当前价格
function updateQuote(data) {
    const price = data.last_price || 10.0;
    currentPrice = price;

    // 更新显示
    document.getElementById('lastPrice').textContent = formatCurrency(price);
    // ...

    // 如果价格输入框为空或用户未修改过，自动填充当前价格
    const priceInput = document.getElementById('price');
    if (!priceInput.hasAttribute('data-user-modified')) {
        priceInput.value = price.toFixed(2);
    }
}

// 监听价格输入框，标记为用户修改
document.addEventListener('DOMContentLoaded', function() {
    const priceInput = document.getElementById('price');
    priceInput.addEventListener('input', function() {
        this.setAttribute('data-user-modified', 'true');
    });
});
```

**验证结果**:
```
  价格输入框 (price): 10.52 ✓
```
