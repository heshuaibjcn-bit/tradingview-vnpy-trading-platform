### Features Implemented:
1. **系统概览**: 显示Agent总数、运行数、健康数、运行时间
2. **Agent列表**: 实时状态、消息数、错误数、运行时间
3. **Agent控制**: 点击Agent卡片查看详情,支持启动/停止/重启
4. **消息流**: 实时显示Agent间通信消息
5. **性能图表**:
   - Message Throughput: 实时消息吞吐量
   - Agent Health Distribution: 健康状态分布
6. **消息历史**: 支持过滤和搜索,显示最近50条消息

### How to Use:
1. 启动交易系统:
   ```bash
   ./venv/bin/python main.py
   ```

2. 打开仪表板:
   ```bash
   ./start-dashboard.sh
   # 或直接打开 dashboard/index.html
   ```

3. 访问 http://localhost:8080

### Notes:
- 仪表板使用纯HTML+JavaScript实现,无需构建
- 通过WebSocket实时接收Agent状态更新
- 通过REST API获取详细数据和执行控制操作
- 支持所有现代浏览器(Chrome 90+, Firefox 88+, Safari 14+)
