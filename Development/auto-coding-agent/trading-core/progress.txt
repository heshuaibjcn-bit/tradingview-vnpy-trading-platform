```bash
  --output messages.json

# 清理旧消息
curl -X POST "http://localhost:8000/api/messages/clean?days=30&archive=false"

# 优化数据库
curl -X POST http://localhost:8000/api/messages/optimize
```

7. **线程安全**: RLock保证并发安全
8. **性能监控**: 实时统计和健康检查

### Statistics Tracked:
- hits - 缓存命中次数
- misses - 缓存未命中次数
- evictions - LRU淘汰次数
- expirations - TTL过期次数
- size - 当前条目数
- total_size_bytes - 总大小估计
- hit_rate - 命中率

### Cache Key Generation:
```python
# Simple key
cache_key("user", 123)  # "user:123"

# With kwargs (sorted for consistency)
cache_key("user", id=123, name="test")  # "id:123:name:test"

# Complex objects (hashed)
cache_key({"data": [1, 2, 3]})  # Hashed for consistency
```

### Cache Invalidation Strategies:
1. **精确失效**: delete(key) - 删除单个条目
2. **模式失效**: invalidate(pattern) - 删除匹配模式的所有条目
3. **全局失效**: invalidate_all(pattern) - 跨所有缓存失效
4. **清空缓存**: clear() - 清空整个缓存
5. **自动过期**: TTL到期自动删除
