# 储能项目投资计算器 - 用户指南

## 系统概述

储能项目投资计算器是一个专业的工商业储能项目投资回报分析工具，帮助用户评估储能项目的经济可行性。

### 核心功能

- **IRR 计算**: 使用 numpy-financial 库进行专业的内部收益率计算
- **NPV 分析**: 基于折现率的净现值分析
- **投资回收期**: 精确到小数年的回收期计算
- **现金流预测**: 15+年的年度现金流预测
- **PDF报告**: 一键生成专业投资分析报告（中文）

## 使用指南

### 第一步：选择投资城市

系统提供5个城市的真实电价数据：

| 城市 | 省份 | 峰时电价 | 谷时电价 | 价差 |
|------|------|----------|----------|------|
| 深圳 | 广东 | 1.20元 | 0.35元 | 0.85元 |
| 广州 | 广东 | 1.15元 | 0.38元 | 0.77元 |
| 佛山 | 广东 | 1.15元 | 0.38元 | 0.77元 |
| 东莞 | 广东 | 1.15元 | 0.38元 | 0.77元 |
| 杭州 | 浙江 | 1.10元 | 0.40元 | 0.70元 |

**选择建议**:
- 价差越大，套利空间越大
- 一线城市（深圳）补贴政策通常更好
- 工业基础雄厚的城市需求更稳定

### 第二步：配置系统规格

#### 装机容量 (MWh)
- **定义**: 储能系统的总储能容量
- **典型值**: 1-50 MWh
- **影响**: 容量越大，总投资越高，但收入也越高
- **建议**: 根据场地条件和预算确定

#### 额定功率 (MW)
- **定义**: 储能系统的充放电功率
- **典型值**: 0.5-20 MW
- **建议**: 功率/容量比通常在 0.5-2.0 之间

#### 充放电效率
- **定义**: 整个充放电循环的效率
- **典型值**: 85-95%
- **影响**: 效率越低，实际可用电量越少
- **技术**: 锂离子电池通常 85-95%

#### 循环寿命
- **定义**: 电池可完成的总充放电次数
- **典型值**: 5000-8000 次
- **影响**: 影响项目寿命期内的更换成本

#### 每日循环次数
- **定义**: 每天完整的充放电循环次数
- **典型值**: 1.0-2.0 次
- **建议**: 根据峰谷时段数量和调度策略确定

### 第三步：设置财务参数

#### 设备成本 (元/MWh)
- **定义**: 储能电池设备的单位成本
- **当前市场价**: 120-180 万元/MWh
- **趋势**: 成本持续下降中

#### 安装成本 (元/MWh)
- **定义**: 系统安装、调试等费用
- **当前市场价**: 20-50 万元/MWh
- **包含**: PCS、BMS、施工等

#### 电网接入成本 (元)
- **定义**: 并网接入、变压器等费用
- **典型值**: 10-100 万元
- **说明**: 与电网距离、容量相关

#### 年维护费 (%)
- **定义**: 每年维护成本占 CAPEX 的百分比
- **典型值**: 1-3%
- **包含**: 检修、更换、运维等

#### 保险费 (%)
- **定义**: 每年保险费占 CAPEX 的百分比
- **典型值**: 0.3-1%
- **建议**: 购买财产险和设备险

#### 项目寿命 (年)
- **定义**: 项目经济分析周期
- **典型值**: 15-20 年
- **影响**: 影响总收益和 IRR 计算

#### 折现率 (%)
- **定义**: 用于 NPV 计算的折现率
- **典型值**: 6-10%
- **说明**: 反映资金的机会成本

### 第四步：选择分析情景

#### 乐观情景
- **假设**: 峰谷价差扩大 10%
- **假设**: 补贴增加 0.1 元/kWh
- **适用**: 电改政策利好时期

#### 基准情景
- **假设**: 基于当前电价政策
- **假设**: 补贴政策不变
- **适用**: 标准投资分析

#### 保守情景
- **假设**: 峰谷价差缩小 10%
- **假设**: 补贴减少 0.05 元/kWh
- **适用**: 风险保守评估

### 第五步：确认信息

系统汇总所有输入信息，请仔细核对：
- 投资城市和电价信息
- 系统技术参数
- 财务参数和总投资
- 分析情景选择

确认无误后点击"开始计算"。

### 第六步：查看结果

#### 关键指标

1. **内部收益率 (IRR)**
   - **定义**: 使项目 NPV = 0 的折现率
   - **判断标准**:
     - IRR > 12%: 强烈推荐
     - IRR > 8%: 可以投资
     - IRR < 8%: 不推荐

2. **净现值 (NPV)**
   - **定义**: 未来现金流的折现总和减去初始投资
   - **判断标准**: NPV > 0 表示项目创造价值

3. **投资回收期**
   - **定义**: 收回初始投资所需时间
   - **判断标准**:
     - < 8 年: 优秀
     - 8-10 年: 可接受
     - > 10 年: 需慎重考虑

4. **投资回报率 (ROI)**
   - **定义**: 总利润 / 总投资 × 100%
   - **判断标准**: ROI > 100% 表示盈利超过投资

#### 年收入构成

- **套利收入**: 低买高卖的差价收入
- **峰谷价差收益**: 削峰填谷的收益
- **补贴收入**: 政府补贴收入

#### 成本分析

- **总投资 (CAPEX)**: 设备 + 安装 + 接入
- **年运营成本 (OPEX)**: 维护 + 保险 + 电费

#### 现金流表

显示未来 15 年（可配置）的详细现金流预测，包括：
- 年份
- 收入（套利、峰谷价差、补贴）
- 成本（维护、保险、电费）
- 净现金流
- 累计现金流

### 生成 PDF 报告

点击"生成 PDF 报告"按钮，系统将生成包含以下内容的中文 PDF 报告：

1. **标题页**: 项目名称、地点、关键参数
2. **执行摘要**: IRR、NPV、回收期等关键指标
3. **系统规格**: 技术参数和电价政策
4. **财务指标**: 详细的财务分析
5. **现金流预测**: 逐年的现金流表格
6. **投资建议**: 推荐/不推荐及理由
7. **风险提示**: 政策、技术、市场、运营风险

报告将自动下载到您的浏览器。

## 计算逻辑说明

### 收入计算

#### 1. 套利收入
```
日套利收入 = 容量 × 循环次数 × 效率 × (峰时电价 - 谷时电价) × 1000
年套利收入 = 日套利收入 × 年运行天数
```

#### 2. 峰谷价差收益
```
年峰谷价差收益 = 功率 (kW) × 100 × 峰谷价值 (CNY/kW/年)
```

#### 3. 补贴收入
```
年补贴收入 = 容量 × 日循环次数 × 效率 × 补贴 × 年运行天数 × 1000
```

### 成本计算

#### 1. CAPEX (资本支出)
```
总投资 = 容量 × (设备单价 + 安装单价) + 电网接入成本
```

#### 2. OPEX (运营支出)
```
年维护费 = 总投资 × 年维护费率
年保险费 = 总投资 × 年保险费率
年电费 = 充电电量 × 谷时电价 × 年运行天数 × 1000
```

### 财务指标

#### IRR (内部收益率)
使用 `numpy_financial.irr()` 计算，求解以下方程：
```
∑(年净现金流 / (1 + IRR)^年) - 初始投资 = 0
```

#### NPV (净现值)
使用 `numpy_financial.npv()` 计算：
```
NPV = ∑(年净现金流 / (1 + 折现率)^年) - 初始投资
```

#### 回收期
```
回收期 = 使累计现金流 ≥ 0 的最小年数
如果年中回收，则按比例计算分数年
```

## 投资建议逻辑

### 强烈推荐
- **IRR > 12%**
- **回收期 < 8 年**
- **NPV > 0**

### 可以投资
- **IRR > 8%**
- **回收期 < 10 年**
- **NPV > 0**

### 不推荐投资
- **IRR < 8%** 或 **回收期 ≥ 10 年** 或 **NPV < 0**

## 常见问题

### Q1: 计算器显示"计算失败"

**原因**: FastAPI 服务未启动或地址配置错误

**解决**:
1. 确认 FastAPI 服务正在运行: `./start-energy-storage-api.sh`
2. 检查 `.env.local` 中的 `FASTAPI_URL` 是否正确
3. 查看 FastAPI 日志确认服务正常

### Q2: 下载 PDF 报告失败

**原因**: FastAPI 报告生成服务出错

**解决**:
1. 确认 FastAPI 服务正在运行
2. 检查 `energy-storage-api/reports/` 目录权限
3. 查看 FastAPI 日志排查错误

### Q3: 计算结果与预期不符

**可能原因**:
- 电价数据不是最新
- 系统参数设置不合理
- 情景假设过于乐观/保守

**建议**:
1. 定期检查政策数据新鲜度（查看监控仪表盘）
2. 调整系统参数到合理范围
3. 尝试不同情景进行敏感性分析

### Q4: IRR 显示为 0 或负数

**原因**: 项目在寿命期内无法收回成本

**建议**:
1. 增大峰谷价差（选择其他城市）
2. 提高系统效率
3. 增加每日循环次数
4. 降低设备成本（等待成本下降）
5. 争取更多补贴

### Q5: 如何选择城市？

**考虑因素**:
- **峰谷价差**: 价差越大，套利空间越大
- **补贴政策**: 一线城市通常补贴更好
- **用电需求**: 工业基础雄厚的城市需求稳定
- **并网条件**: 电网接入的便利性和成本

**推荐城市排序**:
1. **深圳** - 价差大、补贴好、需求强
2. **广州** - 省会城市、政策支持
3. **杭州** - 长三角、数字经济发达
4. **佛山/东莞** - 制造业基地、需求稳定
