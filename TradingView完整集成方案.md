**版本**: v6.0 - TradingView Full Integration
**完成日期**: 2026-03-06
**架构**: 完整TradingView + 量化交易插件

---

## 🎯 设计理念

### 核心原则

1. **保持TradingView完整功能**
   - 使用官方TradingView Widget Library
   - 保留所有原生功能（技术指标、画线工具、多图对比等）
   - 用户可以像使用原生TradingView一样使用图表

2. **量化功能作为插件叠加**
   - 交易面板以侧边栏形式叠加显示
   - 可折叠/展开，不干扰图表使用
   - 独立的选项卡界面（账户、交易、持仓、订单）

3. **无缝集成体验**
   - 插件按钮悬浮在右上角
   - 一键展开/收起交易面板
   - 深色主题与TradingView完美融合

## 🎨 UI/UX设计

### 设计特点

1. **非侵入式设计**
   - 默认隐藏，不干扰TradingView使用
   - 点击按钮才展开
   - 半透明背景，不完全遮挡图表

2. **专业深色主题**
   - 背景色: #131722 (TradingView标准深色)
   - 边框色: #2a2e39
   - 文本色: #d1d4dc
   - 强调色: #2962ff (蓝色)

3. **流畅动画**
   - 侧边栏滑入/滑出: 0.3s ease
   - 按钮悬停: scale(1.1)
   - 状态指示器: 呼吸动画

4. **响应式设计**
   - 桌面端: 380px宽侧边栏
   - 移动端: 100%宽全屏显示
   - 自适应高度

### 交互细节

| 交互 | 效果 |
|------|------|
| 点击插件按钮 | 侧边栏从右侧滑入 |
| 点击关闭按钮 | 侧边栏滑出隐藏 |
| 切换选项卡 | 内容淡入淡出 |
| 悬停按钮 | 放大+阴影增强 |
| Toast通知 | 从顶部滑入，3秒自动消失 |

## 📝 使用说明

### 基本使用流程

1. **打开界面**
   ```
   浏览器访问: http://localhost:8080
   ```

2. **使用TradingView**
   - 直接使用完整的TradingView功能
   - 添加技术指标、画线工具
   - 切换周期和品种

3. **打开交易插件**
   - 点击右上角蓝色插件按钮
   - 侧边栏从右侧滑入

4. **登录账户**
   - 点击"账户"选项卡
   - 如未登录，会弹出登录框
   - 输入用户名和密码

5. **查看账户信息**
   - 总资产、可用资金
   - 持仓市值、今日盈亏

6. **下单交易**
   - 切换到"交易"选项卡
   - 选择买入/卖出
   - 输入股票代码、价格、数量
   - 点击"提交订单"

7. **管理持仓和订单**
   - "持仓"选项卡查看当前持仓
   - "订单"选项卡查看活跃订单
   - 点击"撤单"按钮撤销订单

8. **关闭插件**
   - 点击侧边栏头部的关闭按钮
   - 或再次点击右上角插件按钮

### 快捷操作

| 操作 | 快捷方式 |
|------|---------|
| 展开交易面板 | 点击右上角插件按钮 |
| 收起交易面板 | 点击插件按钮或关闭按钮 |
| 切换选项卡 | 点击选项卡标签 |
| 快速下单 | 交易面板 → 输入价格数量 → 提交 |
| 撤单 | 订单列表 → 点击撤单按钮 |
