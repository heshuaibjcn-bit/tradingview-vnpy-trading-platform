## 🔌 API 接口

### REST API

```
GET  /api/status         # 系统状态
POST /api/connect        # 连接网关
GET  /api/account        # 账户信息
GET  /api/position       # 持仓信息
GET  /api/orders         # 订单列表
POST /api/order          # 发送订单
POST /api/cancel         # 撤销订单
POST /api/subscribe      # 订阅行情
```

### WebSocket 事件

```javascript
socket.on('tick', data)      // 行情推送
socket.on('trade', data)     // 成交推送
socket.on('order', data)     // 订单推送
socket.on('account', data)   // 账户推送
socket.on('position', data)  // 持仓推送
```
